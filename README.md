# aetheris.github.io
Aetheris: The self-evolving Linux AI Agent. Harnessing Federated Learning to solve complex math (IIT-JEE) and system tasks locally. Features a glassmorphism TUI, zero-cost web search, and ad-free Spotify integration. Lightweight enough for i3/2GB RAM, powerful enough to outpace cloud giants. Private, decentralized, and community-trained.
